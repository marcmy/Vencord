# BetterChatCopy

Improves Discord chat copying.

- Makes selected-text context menu copy use the same clean copy path as `Ctrl+C` when possible.
- Adds a message context menu entry for copying nearby conversation text without relying on Discord's virtualized message DOM.
- Can fetch messages up or down from the clicked message, including the clicked message, and preview/edit the text before copying.
- Formats copied text to match Discord's cozy or compact message display setting.
